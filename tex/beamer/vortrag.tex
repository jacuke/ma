\subsection{Fazit}
\frame{
 \frametitle{Fazit}
  \begin{itemize}
  \item Konvertierung von HL7 V2 nach FHIR prinzipiell möglich
  \item Aber unterschiedliche Bearbeitung jedes Nachrichtentyps
  \item Manche Parameter der FHIR Ressourcen ergeben sich lediglich aus dem Kontext der V2 Nachrichten
  \item Mapping der V2 Coding Tables auf FHIR Value Sets teilweise (noch) nicht vorhanden oder inkonsistent
  \item Einige (redundante) Informationen in den V2 Segmenten benötigen nur eine Referenz auf schon bestehende FHIR Ressourcen (z.B. internes Personal/Locations)
  \end{itemize}
}
